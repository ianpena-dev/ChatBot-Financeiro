# 🌐 Interface Web do Chatbot Financeiro

## 🎉 Sua Interface Estilo App Está Pronta!

### 📱 Como Acessar

A interface web está rodando no seu navegador! 

**URL padrão:** `http://localhost:8501`

Se não abrir automaticamente, acesse:
```
http://localhost:8501
```

### ✨ Features da Interface

#### 💬 Chat Estilo App de Mensagens
- **Balões de mensagem** coloridos
- **Você**: Balão azul (direita)
- **Bot**: Balão branco (esquerda)
- **Timestamps** em cada mensagem
- **Scroll automático** para novas mensagens

#### 🎯 Ações Rápidas (Sidebar)
Botões prontos para perguntas comuns:
- 💵 Cálculos Financeiros
- 📚 O que é CDB?
- 🏛️ Tesouro Direto
- 💰 Empréstimo
- 🎯 Metas de Economia
- 📊 Previdência

#### 🎨 Design Moderno
- **Gradientes** bonitos
- **Sombras** suaves
- **Emojis** para visualização
- **Responsivo** e limpo
- **Scrollbar** estilizada

#### 🛠️ Funcionalidades
- **🗑️ Limpar Conversa**: Reseta o chat
- **📥 Exportar Conversa**: Salva em JSON
- **📖 Sobre**: Informações do app

## 🚀 Como Usar

### 1. Acessar a Interface
O navegador deve abrir automaticamente ou acesse:
```
http://localhost:8501
```

### 2. Conversar com o Bot
- **Digite** na caixa de texto na parte inferior
- **OU** clique nos botões de ações rápidas na lateral

### 3. Exemplos de Perguntas
```
💬 "O que é CDB?"
💬 "Quanto rende R$ 10000 a 10% ao ano?"
💬 "Como funciona Tesouro Direto?"
💬 "Quero juntar R$ 100000 em 5 anos"
💬 "Me explique sobre consórcio"
```

### 4. Comandos na Sidebar
- **Limpar Conversa**: Começa do zero
- **Exportar Conversa**: Salva histórico
